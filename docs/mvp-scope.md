# MVP Scope

## Included in the MVP

- User authentication.
- Basic company setup.
- Product catalog.
- Operation catalog.
- Area or workstation catalog.
- Failure mode catalog.
- Quality record entry.
- Basic dashboards.
- CSV data import.
- AI-assisted quality analysis prototype.

## Not Included in the First MVP

- Full ERP integration.
- Full MES integration.
- Real-time machine connectivity.
- Advanced role hierarchy.
- Automated ETL pipelines.
- Native mobile app.
- Full audit compliance engine.

## MVP Principle

The first MVP should avoid heavy IT/OT integrations.

The initial focus is user-uploaded or manually entered data, allowing teams to test the value of structured quality analytics before connecting to complex industrial systems.
